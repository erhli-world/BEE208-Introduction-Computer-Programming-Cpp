Learning Reflection
I have learned and explored several concepts related to programming, software development, electronics, and problem-solving.

In C++, I learned how to use conditional statements such as if, else if, and the ternary operator. I also learned about functions like getline() for accepting user input and pow() for performing exponentiation. While working with development environments such as VS Code and Dev-C++, I encountered errors and learned various troubleshooting techniques to resolve them.

I also gained knowledge about Git and GitHub. I learned the purpose of Git as a version control system and GitHub as a platform for storing and collaborating on code. I practiced uploading files, deleting folders, searching for accounts, and managing file extensions within repositories.

In digital electronics, I explored logic gates including OR, NOT, NAND, and transistor-based circuits. I learned how different logic gates can be combined and implemented on a breadboard. I also worked on understanding circuit diagrams involving LEDs, resistors, push buttons, and transistors.

Additionally, I improved my understanding of measurements and conversions by learning how to interpret tape measure notations and convert feet-and-inch measurements into decimal feet.

Throughout this learning process, I developed stronger problem-solving skills by asking questions, debugging errors, and seeking explanations for concepts that I did not fully understand. Going forward, I plan to continue improving my programming skills, gain more experience with GitHub, and deepen my understanding of electronics and circuit design.
